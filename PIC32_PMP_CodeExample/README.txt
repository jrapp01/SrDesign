	Readme File for Code Example:
--------------------------------------------------

PIC32_PMP\pmp_lcd

--------------------------------------------------
Code Example Description:
--------------------------------------------------
The purpose of this example code is to demonstrate the PIC32MX peripheral library macros and functions supporting PMP module and its various features.

This program writes text to the Explorer-16 LCD display using the PMP library.
